# HackBox
HackBoxは、生活や仕事でのライフハックをシェアするためのサイトです。  

## サイトURL
https://hackbox-app.herokuapp.com

## 主な機能
1. ユーザー登録  
2. ログイン・ログアウト  
3. プロフィール編集  
4. 記事投稿・編集  
5. いいね機能  
6. コメント機能  
7. 投稿の検索機能  
8. ユーザー詳細表示（投稿記事・いいね記事）

## 注目して欲しい点
- ログイン時と未ログイン時で表示を変えた点  
- 実践を意識して、ブランチを切って開発をしている点  

## 使用技術
- 言語/フレームワーク
    - Ruby2.6 / Ruby on Rails 6.0
- 開発環境
    - VS Code
- 本番環境
    - heroku
    - AWS S3
- 使用技術
    - GitHub
